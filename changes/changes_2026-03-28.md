## 1. pesco module additions

### `pesco/pesco/spectral.py` — new public function
- Added `aperiodic_curve(offset, exponent, freqs)`: reconstructs the fixed-mode
  aperiodic component in log10-power space (`offset − exponent × log10(freqs)`).
  Handles both scalar inputs (returns shape `(F,)`) and vectorized arrays of N
  channels (returns `(N, F)`). Replaces the inline `aperiodic_curve_log()` defined
  locally in NB 06.

### `pesco/pesco/stats.py` — new module
- Added `regional_permtest(x, y, n_perm=10_000, seed=None)`: permutation-based
  Spearman correlation for the 38-region cross-modal comparisons, as specified in
  the chapter Methods. Returns a dict with `rho`, `p_perm`, `p_analytic`, `n`, and
  the full null distribution array. Drops NaN pairs before computing.

### `pesco/pesco/peaks.py` — new module
- Added `peak_survival(before_df, after_df, region_col, band_col, present_col)`:
  compares regional peak prevalence (binary) before and after aperiodic correction.
  Returns a DataFrame with `survived`, `lost`, and `gained` columns per region × band.
  Promotes the inline peak-comparison logic from NB 06 into a reusable utility.

### `pesco/pesco/__init__.py` — updated exports
- Added `aperiodic_curve` (from `pesco.spectral`)
- Added `regional_permtest` (from `pesco.stats`)
- Added `peak_survival` (from `pesco.peaks`)

## 4. Literature-driven recommendations (informing the above)

- **Gerster et al. 2022**: specparam absorbs peak power into the aperiodic fit when
  peaks overlap the fitting range; IRASA is less affected. The specparam vs. IRASA
  comparison (NB 09) should report method differences stratified by peak presence,
  not just overall correlations.
- **Donoghue et al. 2021**: R² ≥ 0.90 is the recommended threshold for scientific
  interpretation. Applied consistently now across all notebooks.
- **Tait et al. 2021**: sLORETA (Mantini) vs. eLORETA (Nencki) introduces systematic
  exponent differences — any Mantini–Nencki discrepancy should be attributed to
  inverse-method differences before treating it as a replication failure.
- **Afnan et al. 2023**: the key result to replicate is that cross-modal correspondence
  *improves after aperiodic removal* — this comparison (pre vs. post correction) is
  still missing from NB 05.
